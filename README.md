## Grading

```haskell

-- E1: Exam 1
-- E2: Exam 2
-- A: Assignments

rating = 0.35 E1 + 0.45 E2 + 0.2 A 

grade :: Double -> Char
grade rating
  | rating >= 8.5 = 'A'
  | rating >= 7 = 'B'
  | rating >= 6 = 'C'
  | rating >= 5 = 'D'
  | otherwise = 'F'
```
